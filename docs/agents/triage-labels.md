# Triage labels

This repository uses the default triage label vocabulary.

## Canonical mapping

- `needs-triage` -> maintainer needs evaluation
- `needs-info` -> waiting on reporter
- `ready-for-agent` -> fully specified; AFK-agent ready
- `ready-for-human` -> needs human implementation
- `wontfix` -> will not be actioned

## Guidance

- Apply exactly these strings.
- Do not create alternate spellings unless explicitly reconfigured.
